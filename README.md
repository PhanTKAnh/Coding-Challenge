NAME PROJECT : TASK MANAGER.
Feature Main: 
- Add Task.
- Mark a Task as Complete/Incomplete
- Edit a Task Title
- Delete a Task
- Filter Tasks by Status
Setting:
1. Clone the repository
2. Install dependencies:
- npm install
3. Start the development server:
- npm start
Technology Used
-React.js (v19) 
- React Router DOM (v7) 
- LocalStorage 
-CSS
